# Changelog

## [1.0.0] - 2026-06-13

### Features
- AI-powered travel itinerary generation using Llama 3.3 70B via Groq
- Day-by-day itinerary with morning, afternoon, evening activities
- All prices in ₹ INR — built for Indian travellers
- Budget breakdown: hotel, food, activities, local transport
- Hotel recommendation matched to user preference
- Weather and packing tips for travel month
- Must-try foods as badge pills
- Practical travel tips

### Localization
- Full Hindi (हिन्दी) language support
- Full Telugu (తెలుగు) language support
- AI responses generated in selected language

### AI Backend
- Groq API (Cloud) with BYOK — Bring Your Own Key
- Ollama local inference support (phi3, mistral, tinyllama)
- LM Studio local inference support
- Model selection dropdown

### UX
- User reviews section with star ratings
- Language selector (English / Hindi / Telugu)
- AI settings sidebar
